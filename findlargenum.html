Find The largest Number among the list

Problem Statement

Get 10 Numbers as Input from the user and check each number weather its grater than the next input. if its grater then store it with the variable and repeat for each user input.

Assumption

  • User Should Input Only Numbers.No Alphabet or special Characters allowed
  • Users should enter numbers in the range between 0 to 1000

Pseudocode

  1. Accept a Number from the User
  2. Store the number
  3. Repeat steps 1 and 2 ten Times
  4. For each element check which one is greater and replace the value of greatest variable into greater number
  5. Store the greatest number now you have the greatest number
  6. Print the Number
  7. Print original set of Numbers
