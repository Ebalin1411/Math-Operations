Find The Factorial of the given Number

Problem Statement

Given a number n,Find the Factorial of the Number.

Assumption

  • User Should Input Only Numbers.
  • Users should enter numbers in the range between 0 to 1000

Pseudocode

  1. Accept a Number from the User
  2. Store the number
  3. Formula for all positive numbers :factorial(n)=n∗factorial(n−1)
  4. If the user input is 0 then factorial is 1
  5. Print the Factorial value
