Problem Statement

Get 10 Numbers as Input from the user and display the Sum of the 10 Numbers along with the original set of numbers entered.

Assumption

  • User Should Input Only Numbers.No Alphabet or special Characters allowed
  • Users should enter numbers in the range between 0 to 1000

Pseudocode

  1. Accept a Number from the User
  2. Store the number
  3. Repeat steps 1 and 2 ten Times
  4. Initialize sum to Zero
  5. Take Number and add it to Sum
  6. Take next Number and add it to Sum
  7. Repeat step 6 nine times
  8. Print the value of sum
  9. Print the or original set of Numbers
